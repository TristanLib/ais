# Data Directory

Large raw and processed AIS data files are not stored in Git.

Place downloaded NOAA MarineCadastre.gov AIS source files in `data/raw/` before
running the full high-quality pipeline. Processed arrays will be generated under
`data/processed/`.

See `../DATA_AVAILABILITY.md` and `../REPRODUCIBILITY.md` for source dates,
download notes, and reproduction commands.
